# Examples

This directory contains all example files. Each contains the `main()` entrypoint and thus only one can be compiled at a time. The `/examples` directory is not compiled. However, PlatformIO build environments are defined in `platformio.ini` which can explicitly choose which file in `/examples` is compiled.

See the [`docs/quick_start_guide.md`](/docs/quick_start_guide.md) for information on how to select which of these examples are included during compilation.
